## Criar Nova ADR

1. Copie o padrão de um ADR existente
2. Nomeie como `NNNN-titulo-com-hifens.md`
3. Preencha contexto, opções, decisão e consequências
4. Atualize este índice

## Status

- **Accepted**: Decisão tomada e implementada
- **Proposed**: Em discussão
- **Deprecated**: Não mais relevante
- **Superseded**: Substituída por outra ADR
